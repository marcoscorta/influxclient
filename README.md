# Simple InfluxDB HTML Client

Un client leggero scritto in HTML per connettersi e interagire rapidamente con InfluxDB direttamente dal browser.

## 🚀 Funzionalità

- Connessione tramite **URL**, **Token** e **Organization (Org)**
- Esplorazione dei bucket e dei dati
- Visualizzazione dei dati in formato tabellare
- Generazione di **grafici** per analisi rapide
- **Esportazione dei dati** per utilizzi esterni

## 🧩 Come funziona

Il client non richiede installazione o backend: basta aprire il file HTML nel browser, inserire le credenziali richieste e iniziare a esplorare il database.

## 🔐 Configurazione

Per connettersi sono necessari:

- **URL** del server InfluxDB
- **Token API**
- **Organization (Org)**

> ⚠️ Assicurati che il token abbia i permessi necessari per leggere i dati.

## 📊 Utilizzo

Una volta effettuato l’accesso, è possibile:

1. Navigare tra bucket e misurazioni  
2. Selezionare i dati da visualizzare  
3. Generare grafici direttamente nell’interfaccia  
4. Esportare i risultati  

## 🎯 Scopo

Questo progetto è pensato per:

- Debug rapido  
- Esplorazione dati  
- Analisi veloce senza strumenti complessi  

## 🛠️ Tecnologie

- HTML  
- JavaScript (vanilla)  

## 📌 Note

Questo è un progetto semplice e leggero, pensato per uso pratico e veloce. Non sostituisce strumenti ufficiali o più avanzati, ma offre un’alternativa immediata e accessibile.
